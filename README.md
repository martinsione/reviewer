# reviewer

A TUI for interactively reviewing git diffs with syntax highlighting and hunk staging.

## Install

```bash
bun install
```

## Usage

```bash
bun run dev              # Run in current directory
```

## Keybindings

- `j/k` - Navigate files/hunks
- `s` - Stage hunk
- `u` - Unstage hunk
- `q` - Quit
